# 🐦 Flappy Bird Game (C++)

Flappy Bird Game is a fun arcade-style C++ game inspired by the classic Flappy Bird concept where the player controls a bird and avoids obstacles to survive as long as possible.

## 🚀 Features

* Built using C++ and SFML
* Smooth bird movement and gravity effects
* Real-time obstacle generation
* Collision detection system
* Score tracking
* Endless gameplay mode
* Lightweight and interactive gameplay

## 🕹️ How to Play

1. Start the game
2. Control the bird using keyboard input
3. Avoid hitting pipes and obstacles
4. Pass through gaps to score points
5. Survive as long as possible and beat your high score

## 🎯 Controls

* Spacebar → Make the bird jump
* ESC → Exit Game

## 🛠️ Tech Stack

* C++
* SFML Library
* Object-Oriented Programming
* Game Development Concepts

## ▶️ How to Run

### 1. Clone Repository

```bash
git clone https://github.com/Kushumita/Flappy-Bird-Game.git
```

### 2. Navigate to Folder

```bash
cd Flappy-Bird-Game
```

### 3. Compile

```bash
g++ FlappyBird.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
```

### 4. Run

```bash
./game
```

## 🎯 Objective

Keep the bird flying, avoid obstacles, and score the highest points possible by surviving longer.
